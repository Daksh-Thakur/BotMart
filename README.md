# BotMart

A trendy robotics marketplace landing page built with HTML, CSS, and JavaScript.

## Features

- Product and service listings for robotics sellers
- Category filtering for hardware, services, AI integration, and custom builds
- Contact modal for buyers to message sellers about modifications
- Seller onboarding form for new sellers
- Modern dark-themed UI with glassmorphism styling

## Run locally

Open `index.html` in your browser.

## Files

- `index.html` — page structure and content
- `styles.css` — styles and responsive layout
- `script.js` — client-side marketplace interactions
